Hexo 命令使用

由于每次想更新博客都要重新看一遍 hexo 命令过于麻烦，因此记录一些常用的 hexo 命令和搭建 Hexo 的教程链接。

一、使用 hexo 在 GitHub 上搭建博客

参考链接：[使用 Hexo+GitHub 搭建个人免费博客教程（小白向） - 知乎]:

注意：Repository name 中输入 用户名.github.io ，名称要和用户名一致，比如我这个就只能是 allml777.giyhub.io ，不然无法访问。

二、常用的hexo命令

命令 作用
hexo init 初始化
hexo g 生成页面
hexo s 启动浏览，会在本地构建网页，可以通过访问http://localhost:4000来查看生成页面的效果
hexo d 提交网页到 GitHub
hexo clean 清理缓存
hexo new [layout] <title> 在source文件夹中新建一个markdown文件，就可以使用 Markdown 编辑器在该文件中撰写文章了。layout 可以省略，title用""括起来，示例 hexo new "这次写点什么内容呢"
hexo g -d 就是把 hexo g 和 hexo d 连起来使用

三、一般的更新博客流程

hexo new <title> —> hexo g —>hexo s —> hexo d
新建md文件 —> 根据编写的md文件生成页面 —> 启动页面在本地看看效果 —> 提交到 GitHub
